What is Ratchet?

Ratchet is a light-weight JavaScript MVC framework that helps you build HTML5/JavaScript based applications.

Start by checking out hosted demos

To get a feel for what Ratchet is capable of, check out its hosted examples.

Download Ratchet

Ratchet core library download contains both Ratchet JavaScript library, templating plugins and its supporting libraries. It is available in both minified format (for production) and uncompressed format (for development).

Download for Ratchet examples is a self-contained zip file which includes everything you need to run Ratchet examples locally.

  • Examples  Zip | war

Intro to using Ratchet

Once you've downloaded Ratchet core library, you'll get a zip containing the following

  • ratchet-1.0.0-SNAPSHOT.js
  • ratchet-1.0.0-SNAPSHOT.min.js
  • /plugins/
  • /thirdparty/

Unzip the zip file and then use your favorite text editor to create an empty html file. You will first need to include jQuery library (1.5 or later). 

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>

Now include link to minified Ratchet JavaScript library, minified jQuery Templating Engine plugin and its supporting library in its head section 

<script type="text/javascript" src="ratchet-1.0.0-SNAPSHOT.min.js"></script>
<script type="text/javascript" src="thirdparty/jquery.tmpl.min.js"></script>
<script type="text/javascript" src="plugins/jQueryTemplateEngine.min.js"></script>

Or you can include uncompressed version

<script type="text/javascript" src="ratchet-1.0.0-SNAPSHOT.js"></script>
<script type="text/javascript" src="thirdparty/jquery.tmpl.js"></script>
<script type="text/javascript" src="plugins/jQueryTemplateEngine.js"></script>

If you prefer to use hosted Ratchet library, all you need is to include 

<script type="text/javascript" src="http://code.cloudcms.com/ratchet/1.0.0-SNAPSHOT/js/ratchet-1.0.0-SNAPSHOT.min.js"></script>
<script type="text/javascript" src="http://code.cloudcms.com/ratchet/1.0.0-SNAPSHOT/js/thirdparty/jquery.tmpl.min.js"></script>
<script type="text/javascript" src="http://code.cloudcms.com/ratchet/1.0.0-SNAPSHOT/js/plugins/jQueryTemplateEngine.min.js"></script>

Now add a simple test to the body section of your test html 

<script type="text/javascript">
$(document).ready(function() {
    $('#sample').ratchet(function() {
        this.get("/", function(el) {
            el.html("Ratchet rocks!").swap();
        });
    }).run("/");
});
</script>
<div id="sample"></div>

Save the html file and open it in a browser. You should get a simple page with "Ratchet rocks!".

To learn more about Ratchet, you can also download Ratchet samples and have them running on your machine. Since Ratchet samples contain many examples that use Ajax calls, it is highly recommended to run the examples in a web server (such as Apache Web Server ) or an web application server (such as Tomcat or Jetty).

For web application servers, you can simply download and deploy Ratchet war and then open your browser and point to index.html page of the deployed web application..

For web servers, you need to download and unzip Ratchet samples zip and then open your browser and point to the index.html page which is under the root directory of the unzipped file.

Build Ratchet from source

All Ratchet source code is kept under Git revision control, which you can browse online.

If you have access to Git, you can connect to the repository here: 

git@github.com:gitana/ratchet.git

Once you clone the Ratchet repository to your local machine and have Maven 3.0 or later installed, You can build Ratchet and run Ratchet examples using following Maven command 

mvn clean install jetty:run

Wait till Maven finishes the build process and brings up embedded Jetty server, you should now be able to check out all Ratchet examples from this URL 

http://localhost:8280/examples/index.html
